# bakerventures-site

Company site for Baker Ventures LLC — served at https://bakerventuresstudio.com via GitHub Pages.

- `/` — company one-pager
- `/rizzmaxx/privacy/` and `/rizzmaxx/terms/` — RizzMaxx legal pages (canonical home)
- `CNAME` — custom domain for GitHub Pages

Plain static HTML, no build step. Edit and push to `main` to deploy.
